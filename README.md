
# Therapy App

A comprehensive appointment booking management system with role-based dashboards for administrators, managers, and end users.

## Features

- **Role-Based Dashboards**: Dedicated admin, manager, and user interfaces for managing and booking appointments
- **Google Calendar Integration**: Seamless synchronization with Google Calendar for appointment scheduling
- **Google Authentication**: Secure authentication via Google accounts for mandatory user sign-in
- **Responsive Design**: Built with modern web technologies for optimal user experience

## Tech Stack

- **Backend**: Laravel 12 with Domain-Driven Design (DDD) modular architecture, MySQL
- **Frontend**: React with TypeScript and Tailwind CSS
- **Server Rendering**: Inertia.js for seamless full-stack Laravel + React integration
